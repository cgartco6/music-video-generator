# Custom Voice Directory

Place your custom voice samples here for voice cloning.

## Supported Formats
- WAV (16kHz, mono)
- MP3
- FLAC

## File Naming
- Use descriptive names: `my_singer_voice.wav`
- The name will be used as the voice identifier

## Requirements
- Clean audio with minimal background noise
- 30-60 seconds of voice/singing
- Consistent pitch and tone

## Usage
Upload your voice samples through the web interface, or place them here and they'll be automatically detected.
