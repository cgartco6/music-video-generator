# API Documentation

## Endpoints

### POST /api/upload
Upload audio file for processing.

### POST /api/process
Start video generation with voice cloning and style replication.

### GET /api/status/<job_id>
Check generation status.

### GET /api/download/<filename>
Download generated video.

### GET /api/voices
Get available voices.

### GET /api/styles
Get available music styles.

### POST /api/train_voice
Train a new voice model.

### POST /api/replicate_style
Replicate music style.
